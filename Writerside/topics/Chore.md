# Chore

## Linter

### Setup in your project

```shell
$ rustup component add clippy
```

### Invoke it:

#### From command line:

```shell
$ cargo clippy
$ cargo clippy --fix
```

#### From IDE

* [JetBrains' IDE family](https://github.com/intellij-rust/intellij-rust/issues/4756#issuecomment-565907188)
